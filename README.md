# JDM Car Rental System (React)

A beginner-friendly Car Rental System built using **React**.
This project was originally created as a Python CLI project during school
and later upgraded into a modern frontend application.

---

## Features
- View available cars
- Book a car with name & date
- Booking history
- Data saved using browser localStorage
- Clean component-based structure

---

## Tech Stack
- React (useState, useEffect)
- JavaScript
- HTML & CSS
- Browser localStorage
